## Introdução

### 🚀 **Um companheiro de desktop AI com possibilidades infinitas!**

#### VRM Desktop Pet: Suporta upload de modelos VRM personalizados, ações e cenários 3D para criar um companheiro de desktop personalizado.
![image](doc/image/img-2/vrm.jpeg)

#### Centro de Tarefas: Permite que seus agentes AI executem qualquer tarefa avançada em segundo plano, controlando automaticamente o seu computador para realizar o trabalho, suportando MCP e Agent Skills.
![image](doc/image/img-2/task.jpeg)

#### Chat de Grupo Multi-Papel: Suporta cartões de personagem de tavern e memória de longo prazo, permitindo conversar com vários personagens simultaneamente!
![image](doc/image/img-2/group.jpeg)

#### Bot de Mensagens Instantâneas: Suporta implementação com um clique para QQ, Feishu, DingTalk, Telegram, Discord, Slack
![image](doc/image/img-2/im.jpeg)

#### Bot de Transmissão ao Vivo: Suporta implementação com um clique para Bilibili, YouTube, Twitch e permite transmissão ao vivo panorâmica de 360 graus.
![image](doc/image/img-2/yt.jpeg)

#### Navegador AI: Equipas os teus agentes AI com o seu próprio navegador, suportando controle automático
![image](doc/image/img-2/browser.jpeg)

#### Sistema de Extensões: Suporta instalar extensões e criar novas. A imagem abaixo mostra uma extensão de galgame. Todas as extensões podem ser abertas numa janela independente ou numa barra lateral.
![image](doc/image/img-2/ext.jpeg)

#### Amigo do Programador: Oferece interfaces abertas de API OpenAI e interfaces MCP, permitindo que os agentes sejam conectados externamente
![image](doc/image/img-2/api.jpeg)

#### 🚀 Passos de Utilização

**1. Remover Quaratena de Download de Rede (Importante)**
Após baixar e extrair, abra o Terminal, insira o seguinte comando (note que há um espaço no final), depois arraste a **pasta extraída** para a janela do Terminal e pressione Enter:
```shell
sudo xattr -rd com.apple.quarantine 
```
*(Nota: O parâmetro `-rd` remove recursivamente os atributos de quarentena de todos os componentes dentro da pasta; caso contrário, o ambiente Python pode não funcionar corretamente.)*

**2. Conceder Permissões de Execução de Script**
Navegue para a pasta no Terminal e execute:
```shell
chmod +x quick-update.sh quick-start.sh
```

**3. Executar o Software**
- **Primeira utilização/Atualização:** Recomenda-se executar primeiro `./quick-update.sh` para garantir que as dependências estão sincronizadas com a versão mais recente.
- **Início diário:** Execute diretamente `./quick-start.sh`.

### Docker Deployment (Esta versão do desktop pet só pode ser vista através do navegador.)

- Dois comandos para instalar este projeto:
  ```shell
  docker pull ailm32442/super-agent-party:latest
  docker run -d -p 3456:3456 -v ./super-agent-data:/app/data ailm32442/super-agent-party:latest
  ```

- ⭐Nota! `./super-agent-data` pode ser substituído por qualquer pasta local, após o Docker iniciar, todos os dados serão armazenados em cache nesta pasta local e não serão carregados para nenhum lugar.

- Pronto a usar: aceda a http://localhost:3456/

### Docker Compose Deployment (Esta versão do desktop pet só pode ser vista através de um navegador. Um contentor gateway adicional será iniciado para gestão de login.)

- Instale o projeto:

  ```shell
  git clone https://github.com/heshengtao/super-agent-party.git
  cd super-agent-party
  docker-compose up -d
  ```

- ⭐Nota! O nome de utilizador padrão é `root`, e a palavra-passe padrão é `pass`. Por favor, altere a palavra-passe após o primeiro login.

- Pronto a usar: Visite http://localhost:3456/

- Gestão de API key: Visite http://localhost:3456/token.html

## Utilização

- Desktop: Clique no ícone do desktop para usar imediatamente.

- Web ou docker: Aceda a http://localhost:3456/ após iniciar.

- Chamada de API: Amigo do programador, perfeitamente compatível com o formato OpenAI, pode outputting em streaming, e não afeta a velocidade de resposta da API original. Não precisa de modificar o código de chamada:

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "What is Super Agent Party?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- Chamada MCP: Após iniciar, pode invocar o serviço MCP local escrevendo o seguinte conteúdo no ficheiro de configuração:

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp",
      }
    }
  }
  ```
